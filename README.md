# autotb
## 使用场景
自动复制填表网最左侧列表最新的一条数据,适用于房间难挤的热门房间或放推少的房间,能少操作鼠标两下.体感能比正常操作流程快零点几秒.放推少的房间配合填表网的声音提醒效果更佳.
## 使用方法
具体使用,将程序autotb.exe打开,选好关注BOSS.打开游戏输入ID页面, 有新房间刷新,直接手动粘贴(ctrl+ V),手动进入,手动选召唤即可...
放推较少的BOSS可以配合填表网自带的声音提醒,本程序最小化不影响使用.
