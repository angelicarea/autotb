# autotb
## 使用场景
自动复制填表网最左侧列表最新的一条数据,适用于房间难挤的热门房间或放推少的房间,能少操作鼠标两下.体感能比正常操作流程快零点几秒.放推少的房间配合填表网的声音提醒效果更佳.

## 注意事项
打包后的程序根目录下config.json文件,为更改填表网分支服务器的域名地址配置文件
###结构为
```
{
	"host" : "https://gbf.xinoassassin.me/"
}
```
